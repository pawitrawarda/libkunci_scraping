# Web Scraper - Kunci
Web scraper to get all data from [lib.kunci.or.id](http://lib.kunci.or.id/).
About Kunci: [visit here](http://kunci.or.id/)


# Technologies
 - **NodeJS** version: 10.20.1
 - **Express** version: 4.17.1
 - **Puppeteer** version: 5.3.0
 - **json2csv** version: 5.0.1


# Project Setup

    npm install
    npm run start
